The FuncyStr Library

FuncyStr is a lightweight JavaScript library designed to process strings with embedded functions. It allows dynamic string resolution based on provided parameters, supporting nested and multiple function calls within a single string.

View on GitHub

Available Functions

  • uppercase - Converts text to uppercase
  • repeat - Repeats text N times
  • length - Returns the length of the text
  • pronoun - Returns the appropriate value of gendered language according to chosen pronoun parameter
  • abbrev - Returns an `<abbr>` abbreviation of the term with the description as the title

Function Code

{
    // {{uppercase|text}}: 
    'uppercase': (params, text) => text.toUpperCase(),
    // {{repeat|text|times}}: 
    'repeat': (params, text, times = 2) => text.repeat(parseInt(times)),
    // {{length|text}}: 
    'length': (params, text) => text.length.toString(),
    // {{pronoun|he|she|they}}: 
    'pronoun': (params, he, she, they) => {
        return params.pronoun === 'he' ? he : params.pronoun === 'she' ? she : they;
    },
    // {{abbrev|term|desc}}: 
    'abbrev': (params, term, desc) => {
        // Very basic sanitization of the term and desc, just for the demo
        term = term.replace(/[^a-zA-Z0-9 -]/g, "");
        desc = desc.replace(/[^a-zA-Z0-9 -]/g, "");
        return `${term}`;
    }
}
